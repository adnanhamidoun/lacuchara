🔍 Endpoints Disponibles

GET /health - Health check de la API

GET /restaurants - Obtener lista de restaurantes

POST /predict - Realizar una predicción

POST /predict/menu-items - Predicción de items del menú

GET /cache-stats - Estadísticas del caché

⚙️ Configuración

Para reemplazar este archivo con tu frontend React:

  1. Construye tu frontend: npm run build
  2. Copia los archivos a backend/api/static/
  3. Asegúrate que index.html esté en la raíz de esa carpeta
  4. Reinicia la API - FastAPI servirá automáticamente tu frontend

🐳 Docker

Para desplegar en Docker:

docker-compose up --build - Ejecutar localmente
