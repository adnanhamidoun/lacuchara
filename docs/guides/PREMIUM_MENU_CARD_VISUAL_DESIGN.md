# Premium Menu Card - Visual Design Reference

## Menu Card Visual Design

### Light Mode Color Scheme

```
┌─────────────────────────────────────────────────────────┐
│  Background: Warm cream/ivory gradient                  │
│  #FAF7F0 → #EAE5DB (subtle warm tone)                   │
│                                                          │
│  Accents: Gold (#D4AF37)                                │
│  Text: Dark/Primary (#000 / var(--text))               │
│  Borders: Subtle (#000/20 / var(--border)/40)          │
└─────────────────────────────────────────────────────────┘
```

```
┌─────────────────────────────────────────────────────────┐
│  Background: Deep charcoal gradient                     │
│  #2D2823 → #1F1B16 (deep, warm tone)                   │
│                                                          │
│  Accents: Gold (#D4AF37)                                │
│  Text: Light/Primary (#fff / var(--text))              │
│  Borders: Subtle (#fff/40 / var(--border)/40)          │
```

Used at top of menu card:
```
─────────────────────────────────────────
       ✦  Menú del día  ✦
─────────────────────────────────────────
```

**CSS**:
```tsx
<div className="h-px bg-gradient-to-r from-[#D4AF37] via-[#D4AF37]/40 to-transparent" />
```

### Section Separator
Between menu sections:
```
────────────────────────────────────────
```

**CSS**:
```tsx
<div className="h-px bg-gradient-to-r from-[var(--border)]/20 via-[var(--border)]/40 to-[var(--border)]/20" />
```

At bottom of menu card:
```
        ✦  ✦  ✦
```

```tsx
  <span className="text-lg">✦</span>
```

## Animation Sequence

### Card Entrance
```
Timeline:
0ms    → opacity: 0, y: 20px
600ms  → opacity: 1, y: 0px
```

### Menu Sections (Staggered)
```
Timeline:
Entrantes:    0-500ms     (delay 0ms)
Principales: 50-550ms    (delay 50ms)
Postres:     100-600ms   (delay 100ms)
```

### Menu Items Within Section (Staggered)
```
Item 1: 0-400ms    (delay 0ms)
Item 2: 20-420ms   (delay 20ms)
Item 3: 40-440ms   (delay 40ms)
```

## Customization Guide

### Change the Gold Accent Color
Find all instances of `#D4AF37` and replace with desired color:
```tsx
// Examples:
#C0A080  // Bronze
#B8860B  // Dark goldenrod
#FFD700  // Bright gold
```

### Change Background Colors

**Light Mode**:
```tsx
from-[#FAF7F0]  // Change to preferred cream/ivory
to-[#EAE5DB]    // Change to preferred warm tone
```

**Dark Mode**:
```tsx
from-[#2D2823]  // Change to preferred dark tone
to-[#1F1B16]    // Change to preferred dark tone
```

### Change Typography

**From Serif to Sans**:
```tsx
// Change:
className="font-serif"
// To:
className="font-sans"
```

**Adjust Sizes**:
```tsx
// Restaurant name:
text-3xl  → text-4xl or text-2xl
// Price:
text-4xl  → text-5xl or text-3xl
```

## Design Philosophy

### Why This Aesthetic?

✨ **Premium**: Gold accents and serif fonts elevate the appearance  
📄 **Paper Menu Feel**: Warm background mimics actual menu paper  
🎭 **Editorial**: Serif fonts and deliberate spacing create magazine-like quality  
🏨 **Hospitality**: Elegant and inviting, not technical or clinical  
⚫ **Accessible**: High contrast colors remain readable in any mode  

### Design Principles Applied

1. **Visual Hierarchy**: Size and color guide eye through content
2. **Whitespace**: Generous spacing creates sophistication
3. **Typography**: Serif fonts for elegance, readable sizes
4. **Consistency**: Color palette maintained throughout
5. **Responsiveness**: Scales gracefully on all devices
6. **Motion**: Subtle animations engage without distraction
