   - REGISTRY_USERNAME
   - REGISTRY_PASSWORD

   $ git push origin main

   ✅ Builds image

📋 CHECKLIST Before Deploying:

  □ Docker installed: $ docker --version
  □ Azure CLI installed: $ az --version
  □ Azure account: $ az login
  □ Azure Container Registry created
  □ Azure App Service created
  □ GitHub Secrets configured (if using CI/CD)
  □ BD firewall allows Azure IP ranges
  □ .env NOT in git (check .gitignore)

📊 Image Specifications:

  Frontend:
    • React 18 + Vite + Tailwind CSS
    • Build: npm run build → dist/
    • Size: ~500 KB (gzipped assets)

  Backend:
    • Python 3.10
    • FastAPI + Uvicorn (4 workers)
    • Dependencies: requirements.txt (all pinned)
    • Artifacts: backend/azca/artifacts/*.pkl (ML models)

  Deployment:
    • Base image: python:3.10.11-slim
    • Total size: ~4-5 GB (AzureML dependencies)
    • Security: Non-root user, health checks
    • Ports: 8000 (API), 80/443 (NGINX optional)
    • Resources: Recommended B2+ (Azure)

🔗 Useful Commands:

  # Build locally
  docker build -t azcaapi:latest .

  # Test locally
  docker run -p 8000:8000 azcaapi:latest

  # Push to Azure
  docker tag azcaapi:latest azcaregistry.azurecr.io/azcaapi:latest
  docker push azcaregistry.azurecr.io/azcaapi:latest

  # View Azure logs
  az webapp log tail --name azca-api-app --resource-group azca-rg

  # Restart app
  az webapp restart --name azca-api-app --resource-group azca-rg

  # Scale app
  az appservice plan update --name azca-plan --sku B3
